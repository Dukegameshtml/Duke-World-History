# Duke-World-History
Duke World: Sovereign v4.0 to zaawansowana strategia 4X zamknięta w jednym pliku HTML. Zarządzaj złotem, żelazem i zbożem, by utrzymać armię i zadowolenie trzech frakcji. Eksploruj mapę, odkrywaj technologie i podejmuj trudne decyzje w losowych wydarzeniach. Responsywne UI z trybem mobilnym i systemem zapisu gwarantuje potęgę na lata. 👑🚀
